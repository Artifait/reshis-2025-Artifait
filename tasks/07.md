# Задача 7 — Средний балл на главной странице

## Контекст

Учителя обратились с просьбой улучшить главную страницу дневника. 
Сейчас они видят список учеников с кнопкой «Посмотреть дневник», но, чтобы узнать успеваемость ученика, нужно переходить в его дневник.

Им хочется видеть средний балл сразу на главной странице — рядом с каждым учеником, в его карточке. Это позволит учителям быстро оценить успеваемость класса без лишних кликов.

## Требования

- В карточке каждого ученика на главной странице добавить отображение **среднего балла**.
- Средний балл должен быть рассчитан **по всем предметам** с точностью до **двух знаков после запятой**.
- При отсутствии оценок у ученика показывать текст **«Нет оценок»** вместо числа.
- Средний балл должен быть **визуально заметен**, но не перегружать карточку.
- Функциональность должна работать для ролей: учитель, администратор.

## Пример отображения

```
┌─────────────────────────────┐
│ Иван Петров (10А)           │
│ Средний балл: 4,35          │
│ [Посмотреть дневник]        │
└─────────────────────────────┘

┌─────────────────────────────┐
│ Мария Сидорова (10Б)        │
│ Нет оценок                  │
│ [Посмотреть дневник]        │
└─────────────────────────────┘
```
