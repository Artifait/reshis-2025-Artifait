# Задача 2 — Статистика по оценкам

## Контекст

Хотя ученики нашей школы отлично знают математику и статистику, для удобства было решено показывать основные метрики успеваемости прямо в интерфейсе.

Требования
- Средний балл отображается с точностью **до двух знаков после запятой**.
- Медианный балл рассчитывается по стандартным правилам:
   - Сначала отсортировать массив оценок по возрастанию.
   - Если количество элементов нечетное, медиана — это значение посередине.  
   - Если количество элементов четное, медиана — это среднее арифметическое двух центральных значений.
- Распределение оценок отображается в виде **баров**, где длина бара пропорциональна доле каждой оценки.
- Поле **«Всего оценок»** показывает общее количество выставленных оценок.
- Функция расчета статистики должна корректно работать при **любом количестве оценок** — как при большом, так и при очень малом числе данных.
