# Admin Panel

## Контекст
Админ-панель предназначена для управления пользователями и мониторинга их действий. Основные задачи: удобная модерация, прозрачный аудит, быстрый доступ к ключевым действиям (просмотр Telegram-привязки, проверка активности).

Система включает два блока:
- **Список пользователей (Users list)**
- **Аудит пользователя (User audit)**

Каждый блок реализован так, чтобы:
- интерфейс оставался компактным и отзывчивым,
- таблицы не ломались при больших данных,

## Users List

### Функционал
- Просмотр полного списка пользователей.
- Поиск по username, email или ФИО.
- Быстрые действия по пользователю:
  - открыть аудит,
  - посмотреть статус Telegram-привязки,
  - управление 2FA (просмотр включено/выключено).

## User Audit

### Функционал
- Полный журнал активности пользователя:
  - входы,
  - ошибки входов,
  - смена пароля,
  - попытки привязки/отвязки Telegram,
  - обновления профиля.
- Просмотр деталей каждого события.
- Отключить/включить 2FA
- Открепить Telegram у пользователя
- Потребовать 2FA при следующем входе пользователя

### Структура записи аудита
- `time` — событие в стандартизированном формате.
- `type` — категория (LOGIN, ERROR_LOGIN, PASSWORD_CHANGE и др.).
- `ip` — IP-адрес источника.
- `ua` — User-Agent.
- `details` — JSON или текст с дополнительной информацией.
