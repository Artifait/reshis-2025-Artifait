# Задача 6 — Исправление багов в статистике

## Контекст

В **Задаче 2** ты успешно нашел баги в функциональности статистики по оценкам, теперь стоит их поправить в коде.

## Технические детали

1. Создай **новую ветку** от `feature/02-add-mean-median-score` (ветка с багами из Задачи 2).
2. **Открой список багов** — используйте свои заметки из Задачи 2.
3. **Исправь каждый найденный баг**.
4. **Протестируй исправления** — убедитесь, что все работает корректно.
5. **Создай pull request** с исправленным кодом (из вашей новой ветки).
